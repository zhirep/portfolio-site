Comtutor

A mobile platform designed to find academic help for students.

Project Summary

Comtutor is a mobile application matching students with qualified tutors efficiently. The system matches students and tutors based on subject, grade level, and tutor availability. The app was built and designed using Android Studio using the Kotlin programming language. The project used XML to display the screens and Firebase for its database. This was a project done in around 12 weeks and was completed with 4 other people.

My Role

  • Designed and developed front-end interface of the app.
  • Collaborated with the back-end developers to implement functionalities for some UI aspects.
  • Communicated with the back-end developers to implement logic that were needed for the app.
  • Debugged UI issues that disrupted the user flow.

Technologies Used

  • Kotlin
  • Android Studio
  • XML
