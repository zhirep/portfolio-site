Edmonton Property Search

An application for browsing and filtering real estate listings in Edmonton.

Project Summary

Edmonton Property Search is a responsive web application that allows users to search, filter, and explore property listings within Edmonton. Users can refine results by price range, property type, and location. The project was developed in IntelliJ, along with an API called ArcGIS to display the maps and plot points on it. This was a project done in around 6 weeks and was completed with another person.

My Role

  • Implemented the API as well as utilized the features included in the API, such as the plotting of the points.
  • Developed filtering logic for property search feature.
  • Ensured data are presented clearly.

Technologies Used

  • Java
  • IntelliJ
  • ArcGIS
